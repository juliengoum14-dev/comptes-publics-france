# Source FIPECO

## Métadonnées

- **URL de base** : https://www.fipeco.fr
- **Formats disponibles** : PDF / HTML
- **Périmètre** : APU (État + SS + Collectivités)
- **Granularité** : par nature, par fonction, par catégorie d'APU
- **Périodicité** : annuelle + actualité
- **Séries disponibles** : depuis 1975 (dépenses), depuis 1978 (par catégorie APU)
- **Licence** : ouverte (gratuite)
- **API disponible** : non
- **Fiabilité** : source secondaire de référence (ex-DG Trésor)
- **Dernière mise à jour** : mars 2026
- **Facilité d'accès** : téléchargement PDF direct

## Contenu

- **Fiches encyclopédiques** : 50+ fiches structurées par thème
- **Notes d'analyse** : études approfondies
- **Graphiques et tableaux** : visualisations interactives
- **Données historiques longues** : dépenses 1975-2024, séries longues

## Thèmes couverts

- Les comptes publics (définitions, périmètres)
- Déficit, dette et politique budgétaire
- Programmation et pilotage des FP
- Prélèvements obligatoires
- Dépenses publiques
- Masse salariale publique
- Finances locales
- Assurances sociales et redistribution

## Observations

- Excellente source pour comprendre la structure des FP
- Données historiques longues très utiles
- Pas de données brutes téléchargeables (PDF uniquement)
- Analyse pédagogique de qualité
- Mise à jour régulière
