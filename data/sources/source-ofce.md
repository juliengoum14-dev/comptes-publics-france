# Source OFCE

## Métadonnées

- **URL de base** : https://www.ofce.sciences-po.fr
- **Formats disponibles** : HTML / PDF
- **Périmètre** : APU (prévisions et analyses)
- **Granularité** : par sous-secteur, par nature de dépense/recette
- **Périodicité** : trimestrielle (prévisions avril + octobre)
- **Séries disponibles** : prévisions N à N+2
- **Licence** : ouverte
- **API disponible** : non
- **Fiabilité** : source académique de référence
- **Dernière mise à jour** : avril 2026 (prévisions)
- **Facilité d'accès** : accès web direct

## Publications clés

| Publication | Contenu |
|---|---|
| Prévisions OFCE (avril) | Panorama complet macro + finances publiques |
| Policy brief | Analyses thématiques (ex: Budget 2026) |
| Notes de conjoncture | Analyses trimestrielles |
| Analyses budget | Impact macroéconomique du budget |
